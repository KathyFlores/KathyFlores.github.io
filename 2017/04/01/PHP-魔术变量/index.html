PHP魔术变量

魔术变量

PHP在设计的时候已经预定义了9个超级全局变量、8个魔术变量和13魔术函数，这些变量与函数可以在脚本的任何地方使用（不用预先声明）。魔术变量不区分大小写。

1 __LINE__

文件中的 当前行号 。
EXAMPLE:

<?php
echo '这是第 " '  . __LINE__ . ' " 行';
?>

OUTPUT: 这是第 “ 2 “ 行

2 __FILE__

文件的 完整路径 和 文件名 。
若用在被包含文件中，则返回被包含的文件名。
NOTE: 超全局变量$_SERVER[‘PHP_SELF’]只返回文件名，而无绝对路径。
EXAMPLE:

<?php
echo '该文件位于 " '  . __FILE__ . ' " ';
?>

OUTPUT: 该文件位于 “ D:\php-7.1.3-src\Apache24\htdocs\getuser.php “

3 __DIR__

文件 所在的目录 。
若用在被包括文件中，则返回被包括的文件所在的目录。
等价于dirname(__FILE__)。
NOTE：不包括末尾的斜杠，除非是根目录。
EXAMPLE:

<?php
echo '该文件位于 " '  . __DIR__ . ' " ';
?>

OUTPUT: 该文件位于 “ D:\php-7.1.3-src\Apache24\htdocs “

4 __FUNCTION__

函数名称 （区分大小写）。
EXAMPLE:

<?php
function test() 
{
    echo  '函数名为：' . __FUNCTION__ ;
}
test();
?>

OUTPUT: 函数名为：test

5 __CLASS__

类的名称 （区分大小写）。
NOTE:

  • 类名包括其被声明的作用区域。
  • 对trait也起作用，在trait方法中使用，返回调用trait的类的名字。

EXAMPLE:

<?php
class test 
{
    function _print() 
    {
        echo '类名为：'  . __CLASS__ . "<br>";
    }
}
$t = new test();
$t->_print();
?>

OUTPUT: 类名为：test

6 __TRAIT__

Trait的名字 。
NOTE: trait名字包括其被声明的作用区域。
EXAMPLE:

<?php
class Base 
{
    public function sayHello() 
    {
        echo 'Hello ';
    }
}
trait SayWorld {
    public function sayHello() 
    {
        parent::sayHello();
        echo 'World!'.__TRAIT__;
    }
}
class MyHelloWorld extends Base 
{
    use SayWorld;
}
$o = new MyHelloWorld();
$o->sayHello();
?>

OUTPUT: Hello World!SayWorld

7 __METHOD__

方法名 ，与__FUNCTION__类似。

8 __NAMESPACE__

当前命名空间的名称 ，在编译时定义。
EXAMPLE:

<?php
namespace MyProject;
echo '命名空间为："', __NAMESPACE__, '"';
?>

OUTPUT: 命名空间为：MyProject
